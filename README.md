# PriceWatch
A web application for monitoring product prices across Romanian online stores. Built with Java and Jakarta EE, it automatically tracks price history and notifies users by email when a watched product drops below a target price. Developed as a bachelor's thesis project.
 
## Tech Stack
* **Language:** Java
* **Framework:** Jakarta EE 10 (Servlets, JSP, EJB Stateless Beans)
* **Application Server:** GlassFish 7
* **Database:** MySQL (JPA)
* **Web Scraping:** Playwright
* **Email Notifications:** JavaMail (Gmail SMTP)
* **Frontend:** JSP, Tailwind CSS, Chart.js, Lucide Icons
* **Authentication & Authorization:** JDBC Realm (SHA-256 hashing), role-based access control
